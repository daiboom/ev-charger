# ev-charger
🔌 전기차 충전기 사용자 인터페이스 - Rust와 EGUI로 개발한 터치스크린 기반 GUI 애플리케이션. 산업용 전기차 충전 시스템을 위한 실시간 모니터링 및 제어 기능 제공
